# 喵彩 ❤️

从我们开始养猫到现在一直用的一款猫砂

也是两只猪咪从小用到现在的一款
对比其他猫砂来说，平价
是真的除臭，不会让家里有特别大的异味

两只猫，大概两天铲一次

喵彩淘宝直达 (opens new window)

PS: 每次我都是一批一批的屯...

# 好主人 ❌

我不知道别人用这款猫砂是什么情况
但是！我真的是受不了了！是真的臭！！！
而且还是整个房间都在臭！无与伦比的臭！！！

即使是新的倒进去，猫咪一小便
那个说不上来的臭味就飘起来了，哎
而且即使埋上了还是很大的臭味

两只猫，几乎每次它们上完厕所就得铲...

结团快，而且不容易散！！！

遮臭能力一般~

两只猫，平均两天一铲~

# 雪貂留香 ✔️

这款的效果和福丸的差不多
有一丝绿茶的清香，但使用过后味道就变得有些奇怪了
也不能说是臭，也不是香，就是有些独特吧

我比较习惯用这款的和其他猫砂混着
