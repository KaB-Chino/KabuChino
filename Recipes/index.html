# 前言

首先，我是一个不是很懂猫咪饮食的人
也在慢慢的学习科学养猫

但我更喜欢把一切关于猫的问题回归到自然状态
也就是野外的猫是如何生活的
毕竟我家的两只都是流浪橘猫的后代

首先考虑一个问题：
自然界的猫以什么为食？

猫通常是捕食一些小型的啮齿类动物

比如老鼠等，捕到猎物，猫吃的是内脏、肌肉、骨肉以及一些皮毛

但经过人类的漫长驯化
有极大一部分的猫成了家猫、宠物猫

宠物猫
一些主人可能在给猫吃的这方面会特别好
低纤维 高脂肪

家猫
可能主人就会把一些米饭、白粥、鱼骨以及一些人所吃剩下的肉类给猫咪来吃
高纤维 高盐 低蛋白

# 我の猫粮

刚开始第一款给他们两只吃的是 吉米高幼猫猫粮open in new window
过了一段时间才了解到，这款猫粮各项指标都太低了

通过其他平台得知了一款名为 麦富迪 的猫粮
我们从他们三个月大左右就给他们换成了成年猫猫粮

一直吃到今年4月份 我家奇诺的肠胃很脆弱的样子 一有点不适应就会吐，又或者软便 加上橘猫卡布的爪子也出现了发白

通过询问才得知，这是已经缺乏营养的现象 并且得知，麦富迪open in new window 这款猫粮的国标指标是刚好达标的

依稀还记得
在2月底的时候，刚好那几天天气变化无常
奇诺的抵抗力低，不吃东西，一吃就吐
每天晚上凌晨两三点就开始吐，还伴随着发烧

大半夜去询问了一下宠物店的姐姐
被告知，可以吃点抗生素或喝点纯羊奶

第二天带奇诺去打了一针并拿了点消炎药以及纯羊奶
辗转反复两三天，才能睡个安稳觉


# 现在吃的猫粮

现在给他们吃的是 WoWoopen in new window

这款猫粮刚开始接触是因为隔壁养猫的因为猫咪去喵星了
刚好有一袋还没拆封的 WoWo猫粮 说给我家猫试试

WoWo这款猫粮的气味挺香的
有一股淡淡的玉米香味，有一些海苔碎以及冻干粒

但感觉 WoWo 会比较油，也容易黑下巴，也会容易上火易多眼屎

# 江小傲

这一款是女朋友在养猫群里，一个猫友极力推荐的
抱着试一试的心态，买回来一斤的试用装

结果就是，两只猪并不是很爱吃
就比如平时倒粮，两只猪都是飞奔过来吃的
但放的是江小傲的，就闻了下就走了

并且会软便！！而且奇诺还吐了一次
所以第二天就没给他们吃了

# 麦富迪

对比了当时所接触到的每一种猫粮
麦富迪会脱颖而出是因为他们家的 冻干open in new window

我们是先买了他们家的冻干，才接触到这一款的猫粮
当时买回来的冻干，我家的橘猪卡布是绞尽脑汁偷吃冻干
当时盖子都给我咬烂了，偷吃了一大半瓶的冻干

但猫粮的营养就不是很好了
因为这款猫粮的粗纤维成分太高了
长期食用，对猫咪各方面发育都会出现问题
就好比如缺钙指甲发白，肠胃脆弱，抵抗力差
